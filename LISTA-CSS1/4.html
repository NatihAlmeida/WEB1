Formas de aplicar CSS no documento HTML:

CSS Inline:
Inline é uma forma direta de aplicar CSS no documento HTML. (Este paragrafo foi estilizado através do CSS Inline com a cor roxa.)

CSS Interno ou Incorporado:
É como se fosse aplicar o CSS através de uma sessão, no caso é aplicado através da tag style dentro da tag head. (Este paragrafo foi estilizado através do CSS Interno especificado dentro da tag head com cor: goldenrod e fonte: times new roman.)

CSS Externo:
E este aqui, o CSS é aplicado de maneira externa através de um link que direciona para um arquivo externo, ou seja, é aplicado de forma separada no arquivo externo. (Este paragrafo foi estilizado através do CSS Externo especificado através do link na tag head que referencia o arquivo externo 4.css).
