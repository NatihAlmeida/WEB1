As formas de aplicar e exemplos dos 3 tipos de incorporação do CSS estão nos arquivos 4.html e 4.CSS.

O exercício 6 é semelhante ao exercício 4, com uma pergunta diferente que é sobre o impacto de cada metódo de desempenho e na organização do código referente aos diferentes tipos de aplicação do CSS - Inline, Interno e Externo.


Respondendo sobre o impacto de cada metódo:  

CSS Inline que é uma forma dirente de aplicar estilização no documento HTML é fácil e rápido para pequenas alterações em um ou outro elemento de um projeto pequeno. Agora se for para um projeto grande não é tão legal, porque o documento HTML vai ficar pesado e massante.


Referente ao CSS Interno que é aplicado através da tag <style> dentro do <head>. Para projetos pequenos com uma página é interessante e viável, pois é uma forma rápida de estilizar vários elementos num um único documento HTML. Já para projetos maiores, com duas ou mais páginas por exemplo não é bom. Então esse tipo de incorporação Interno não serve para ser utilizado em outras páginas porquê se torna complicado e pode deixar as páginas mais lentas.


E o CSS Externo, é útil e muito bom, pois é aplicado em um arquivo externo facilitando por exemplo: a estilização de uma página específica ou diversos elementos em diferentes páginas, além de que pode ser reutilizado e contribui para o melhor desempenho de várias páginas.
